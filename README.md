Acest proiect implementează și prezintă filtre RC trece-jos (FTJ) și trece-sus (FTS) utilizând o interfață MATLAB. Aplicația permite vizualizarea comportamentului circuitelor și analizarea răspunsului în frecvență (amplitudine și fază) pentru fiecare tip de filtru.

Funcționalități principale:
Meniul principal este deschis prin fișierul pornire.m.
Butonul Interfață deschide fișierul start_int.m, care apelează funcția Interfata pentru navigare.
Butonul FIF (circuit la înaltă frecvență) deschide o interfață dedicată.
Butonul Grafice încarcă fișierul semnal.m, care apelează funcția grafice și afișează caracteristica de frecvență/fază pentru filtrul RC trece-sus (FTS).
Butonul Grafice FTJ afișează caracteristica de frecvență/fază pentru filtrul RC trece-jos (FTJ).
